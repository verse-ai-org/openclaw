## 简介

OpenClaw 是一款可在本地设备上运行的个人 AI 助手，支持多通道消息集成（如 WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、iMessage、BlueBubbles、IRC、Microsoft Teams、Matrix、Feishu、LINE、Mattermost、Nextcloud Talk、Nostr、Synology Chat、Tlon、Twitch、Zalo、Zalo Personal、WebChat 等），并提供语音唤醒、实时画布、浏览器控制、节点工具、自动化任务等能力。本“快速开始”旨在帮助你在最短时间内完成安装、配置与首次使用。

## 系统要求

- 运行时：Node.js 版本需满足最低要求（详见下文）
- 操作系统：macOS、Linux 或 Windows（推荐使用 WSL2）
- 包管理器：从源码构建时需要 pnpm；使用 npm/pnpm/bun 安装时可直接进行全局安装

关键要点

- Node 最低版本要求：≥ 22.12.0
- 推荐使用 npm、pnpm 或 Bun 进行安装（仓库文档明确支持这三种方式）
- Windows 平台强烈建议通过 WSL2 使用

## 常见问题与故障排除

- 命令未找到（openclaw: command not found）

  - [docs/install/node.md: 第91–126行:91-126](file://docs/install/node.md#L91-L126)
  - [docs/install/index.md: 第181–204行:181-204](file://docs/install/index.md#L181-L204)

- 权限错误（Linux）
  - 将 npm 全局前缀改为用户可写目录并更新 PATH
  - [docs/install/node.md: 第128–139行:128-139](file://docs/install/node.md#L128-L139)

- sharp 构建失败（macOS）
  - 强制使用预编译二进制或安装构建工具
  - [docs/install/index.md: 第82–90行:82-90](file://docs/install/index.md#L82-L90)

- 构建工具缺失（Linux/macOS）
  - 安装 build-essential、python3、make、g++、cmake 等工具
